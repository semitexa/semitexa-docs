# Module Structure

This file is a project-level wrapper.

The canonical Semitexa module layout lives in:

- `packages/semitexa-core/docs/MODULE_STRUCTURE.md`

Use this root file only as a stable entrypoint for project and AI navigation.

## Project Rule

For Semitexa projects in this monorepo, module structure must follow the canonical layout defined by `semitexa-core`.

If project-specific constraints are ever added here, they must extend the core rules instead of restating or forking them.
