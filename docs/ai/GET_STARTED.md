## Agent Checklist

1. Get or create the project.
2. Use `curl -fsSL https://semitexa.com/install.sh | bash -s my-project` for a new local install.
3. `cd my-project`
4. Copy `.env.example` to `.env`.
5. Start with `bin/semitexa server:start`.
